# 设置着色模式

## 功能说明

支持设置泳道图的着色模式，例如根据子图或节点类型等对泳道图进行颜色区分，以帮助用户更方便地识别图中的信息。

## 前提条件

执行PyPTO程序，生成泳道图文件，并且在同级目录下存在dyn\_topo.txt文件。

## 操作步骤

1.  打开泳道图文件。以下提供两种方式：
    1.  在Visual Studio Code侧边栏，单击![](../figures/zh-cn_image_0000002491265870.png)图标，打开PyPTO Toolkit，在运行结果界面打开泳道图文件。
    2.  在Visual Studio Code工作区，右键单击泳道图文件，在弹出的菜单中选择“PyPTO Toolkit：打开文件”。

2.  在泳道图界面中，单击右侧的![](../figures/zh-cn_image_0000002489256878.png)按钮。

    ![](../figures/zh-cn_image_0000002502052124.png)

3.  根据界面展示进行图层配置。

    ![](../figures/zh-cn_image_0000002533812261.png)

    **表 1**  配置说明

    |参数|说明|
    |--|--|
    |着色模式|泳道图上节点的着色模式，取值包括：按子图：根据不同的子图显示不同的颜色。默认展示模式。按Stitched Function：根据不同的缝合函数显示不同的颜色。按语义标签：根据不同的计算类型显示不同的颜色。|
    |Task内容展示|通过该选择配置泳道图节点的名称，取值包括：</br>任务名称：展示任务名称，例如：0-10-12-0-0。</br>语义标签：展示计算类型，例如Matmul。</br>全部勾选则全部展示，例如：0-10-12-0-0（Matmul）。|
    |任务连线层级|通过该配置显示泳道图节点的连线层级。例如：配置为-1表示展示该节点的全部层级，配置为1表示展示该节点的前后一层依赖关系。|


4.  配置效果展示。

    例如“着色模式”配置为“按语义标签”，“Task内容展示”配置为“语义标签”后，泳道图界面展示如下。
