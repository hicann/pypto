# 测量节点间的时间间隔

## 功能说明

支持测量泳道图两节点间的时间间隔，当前支持自动测距和手动测距。

-   自动测距：只支持测量同一泳道内相邻节点间的时间间隔。
-   手动测距：支持跨泳道、跨节点测量两节点间的时间间隔。

## 前提条件

执行PyPTO程序，生成泳道图文件。

## 操作步骤

1.  打开泳道图文件。以下提供两种方式：
    -  在Visual Studio Code侧边栏，单击![](../figures/zh-cn_image_0000002491265870.png)图标，打开PyPTO Toolkit，在运行结果界面打开泳道图文件。
    -  在Visual Studio Code工作区，右键单击泳道图文件，在弹出的菜单中选择“PyPTO Toolkit：打开文件”。

2.  自动测距。

    将鼠标悬浮到面板中的两个节点之间，自动显示两节点间的时间间隔。

    ![](../figures/zh-cn_image_0000002533650005.png)

3.  手动测距。

    长按Alt键，依次单击起始节点，自动显示两节点间的时间间隔。

    ![](../figures/zh-cn_image_0000002533810067.png)
