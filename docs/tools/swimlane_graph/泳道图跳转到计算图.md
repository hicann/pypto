# 泳道图跳转到计算图

## 功能说明

支持从泳道图的某个Task节点跳转到对应的计算图，方便用户进行问题定位。

## 前提条件

执行PyPTO程序，生成计算图文件和泳道图文件。

## 操作步骤

1.  打开泳道图文件。以下提供两种方式：
    -  在Visual Studio Code侧边栏，单击![](../figures/zh-cn_image_0000002491265870.png)图标，打开PyPTO Toolkit，在运行结果界面打开泳道图文件。
    -  在Visual Studio Code工作区，右键单击泳道图文件，在弹出的菜单中选择“PyPTO Toolkit：打开文件”。

2.  单击泳道图中某个节点，拉起节点详情面板。![](../figures/zh-cn_image_0000002533806679.png)
3.  单击“绑定计算图”，选择program.json文件绑定该文件。

    若merged\_swimlane.json和program.json在同级目录下，会进行自动绑定。

    ![](../figures/zh-cn_image_0000002517774919.png)

4.  单击相关字段即可跳转到对应的计算图。

    ![](../figures/zh-cn_image_0000002517780081.png)

    单击rootHash值，可跳转到该任务对应的Execute Graph，便于查看该任务的前后依赖关系。进而通过callOpMagic可定位到Call Operation节点，即该任务对应的Block Graph。

    单击leafHash值，可直接跳转到该任务对应的Block Graph。
