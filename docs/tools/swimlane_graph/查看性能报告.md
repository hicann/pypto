# 查看性能报告

## 功能说明

支持查看汇总性能数据，支持按全局，时间范围、区域及任务节点查看。

## 前提条件

执行PyPTO程序，生成泳道图文件。

## 操作步骤

1.  打开泳道图文件。以下提供两种方式：
    -  在Visual Studio Code侧边栏，单击![](../figures/zh-cn_image_0000002491265870.png)图标，打开PyPTO Toolkit，在运行结果界面打开泳道图文件。
    -  在Visual Studio Code工作区，右键单击泳道图文件，在弹出的菜单中选择“PyPTO Toolkit：打开文件”。

2.  查看全局级性能数据。

    单击泳道图右侧的![](../figures/zh-cn_image_0000002521522013.png)按钮。

    ![](../figures/zh-cn_image_0000002502051718.png)

    页面右边弹出性能数据统计面板。

    ![](../figures/zh-cn_image_0000002502428688.png)

    **表 1**  参数说明

    |参数|说明|
    |--|--|
    |Task Time|执行总时间，即第一个Task执行开始到最后一个Task执行完的时 间。|
    |iCache命中率（暂不支持）|iCache缓存命中率。|
    |AICore Time|所有AI Core泳道中的任务总耗时。|
    |AICore利用率|AI Core Time / (时间轴总时长 * AICore泳道的个数)。|
    |Cube Cycles（暂不支持）|Cube占用的时钟周期。|
    |Vector Cycles（暂不支持）|Vector占用的时钟周期。|


3.  查看某时间范围内的性能数据。

    详细操作请参见[按时间范围查看泳道图](按时间范围查看泳道图.md)。

4.  查看某区域内的性能数据。

    在泳道图界面单击并拖拽出度量区域，下方展示该区域的性能数据。

    ![](../figures/zh-cn_image_0000002534229551.png)

    详细参数解释请参见[按时间范围查看泳道图](按时间范围查看泳道图.md)。

5.  查看某节点的性能数据。

    详细操作请参见[查看泳道图](查看泳道图.md)。
