# 搜索计算图节点

## 功能说明

打开计算图后，可以针对计算图中的节点信息进行搜索。支持如下搜索模式：

-   当前图搜索（默认）：仅搜索当前的计算图的节点信息。
-   整图搜索：对该计算图关联的所有子图进行搜索。
-   仅通路搜索：当只看通路或锁定通路时触发，只搜索当前锁定的这条路径的节点信息。

此外，还支持根据按照特定的标签进行精准和模糊搜索。

## 前提条件

执行PyPTO程序，生成计算图文件。

## 操作步骤

1.  打开计算图文件，下面提供两种方式，用户任选其一即可：
    -   在Visual Studio Code侧边栏，单击![](../figures/zh-cn_image_0000002491082472.png)图标，打开PyPTO Toolkit，在运行结果界面打开计算图文件\(此种方式仅支持打开文件命名以After开头的图\)。
    -   在Visual Studio Code工作区，右键单击计算图文件，在弹出的菜单中选择“PyPTO Toolkit：打开文件”。

2.  按照不同的搜索模式进行搜索。
    -   当前图搜索：在顶部搜索栏选择“当前图”，输入搜索关键词，单击搜索结果可跳转至当前图的对应节点。

        ![](../figures/zh-cn_image_0000002512352587.png)

    -   整图搜索：在顶部搜索栏选择“整图”，输入搜索关键词，单击搜索结果可跳转至对应子图。

        ![](../figures/zh-cn_image_0000002480352670.png)

    -   仅通路搜索：锁定某条链路，具体操作请参见[锁定计算链路](锁定计算链路.md)，输入搜索关键词，单击搜索结果可跳转至当前图的对应节点。

        ![](../figures/zh-cn_image_0000002480192706.png)

3.  按照指定标签进行搜索。在顶部搜索栏进行如下设置。

    ![](../figures/zh-cn_image_0000002512312555.png)

    用户可以选择父标签和子标签进行准确搜索，也可以不指定子标签进行模糊搜索。模糊搜索时可以指定是否区分大小写和是否使用正则表达式，如下图所示：

    ![](../figures/zh-cn_image_0000002512316779.png)
