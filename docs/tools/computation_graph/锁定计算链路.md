# 锁定计算链路

## 功能说明

支持锁定当前通路并高亮显示，便于用户仅查看当前通路的信息。

## 前提条件

执行PyPTO程序，生成计算图文件。

## 操作步骤

1.  打开计算图文件，下面提供两种方式，用户任选其一即可：
    -   在Visual Studio Code侧边栏，单击![](../figures/zh-cn_image_0000002491082472.png)图标，打开PyPTO Toolkit，在运行结果界面打开计算图文件\(此种方式仅支持打开文件命名以After开头的图\)。
    -   在Visual Studio Code工作区，右键单击计算图文件，在弹出的菜单中选择“PyPTO Toolkit：打开文件”。

2.  右键单击计算图中的节点，或鼠标悬浮到节点上并通过快捷键（Ctrl+H）选择“锁定当前通路并高亮”。

    ![](../figures/zh-cn_image_0000002502414338.png)

    当前链路整体高亮锁定，其他节点置灰。通过快捷键（ESC）可退出锁定。

    ![](../figures/zh-cn_image_0000002480215508.png)

3.  右键单击计算图中的节点，或鼠标悬浮到节点上并通过快捷键（Ctrl+H）选择“只看当前通路”。

    当前链路整体高亮锁定，其他节点消失。通过快捷键（ESC）可退出锁定。

    ![](../figures/zh-cn_image_0000002512495361.png)
