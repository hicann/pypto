# 局部搜索渲染计算图

## 功能说明

针对超大规模数据文件，由于节点数量较多，可能会导致打开速度过慢或无法打开。通过搜索特定的节点来局部渲染计算图，这样可以清晰展示特定链路，有效解决加载延迟问题并提高交互效率。目前提供两种操作方式，用户任选其一即可。

## 操作步骤（方式一）

1.  启动局部渲染模式，以下入口任选其一。
    -  在Visual Studio Code侧边栏，单击![](../figures/zh-cn_image_0000002532734371.png)图标，打开PyPTO Toolkit，在运行结果界面打开计算图文件。
    -  在Visual Studio Code工作区，右键单击计算图文件，在弹出的菜单中选择“PyPTO Toolkit：打开文件”。

2.  当打开节点数超过10000的计算图文件时，显示以下界面。当未超过10000节点时，则不会显示以下界面，直接全量渲染计算图。

    ![](../figures/zh-cn_image_0000002484263090.png)

3.  选择“搜索模式”，显示以下界面。

    ![](../figures/zh-cn_image_0000002500707780.png)

4.  搜索“当前图”或者“整图”中的某个节点，计算图将根据节点的前后关联链路进行局部渲染。

    **图 1**  基于当前图渲染  
    ![](../figures/rendering_based_on_current_graph.png)

    **图 2**  基于整图渲染  
    ![](../figures/rendering_based_on_entire_graph.png")

5.  单击搜索结果列表，即可进入该节点相关路径的渲染。同时插件将记录本文件的本次搜索结果，下次打开本文件搜索界面时，单击历史记录即可渲染计算图。

    ![](../figures/zh-cn_image_0000002532707651.png)

## 操作步骤（方式二）

1.  启动局部渲染模式。

    在Visual Studio Code工作区，右键单击计算图文件，在弹出的菜单中选择“PyPTO Toolkit：高级操作”，在二级菜单中选择 “PyPTO Toolkit：按节点搜索打开计算图”。

2.  显示以下界面。

    ![](../figures/zh-cn_image_0000002532704255.png)

3.  “当前图”或者“整图”中某个节点，计算图将根据节点的前后关联链路进行局部渲染。

    **图 3**  基于当前图渲染  
    ![](../figures/rendering_based_on_current_graph_0.png)

    **图 4**  基于整图渲染  
    ![](../figures/rendering_based_on_entire_graph_1.png)

4.  单击搜索结果列表，即可进入该节点相关路径的渲染。同时插件将记录本文件的本次搜索结果，下次打开本文件搜索界面时，单击历史记录即可渲染计算图。

    ![](../figures/zh-cn_image_0000002500707666.png)
