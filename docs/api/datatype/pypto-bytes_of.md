# pypto.bytes\_of

## 产品支持情况

| 产品             | 是否支持 |
|:-----------------|:--------:|
| Atlas A3 训练系列产品/Atlas A3 推理系列产品 |    √     |
| Atlas A2 训练系列产品/Atlas A2 推理系列产品 |    √     |

## 功能说明

返回某个数据类型所占的bytes的大小。

## 函数原型

```python
bytes_of(dtype: pypto.DataType) -> int
```

## 参数说明


| 参数名 | 输入/输出 | 说明                              |
|--------|-----------|-----------------------------------|
| dtype  | 输入      | 需要查看bytes大小的数据类型。     |

## 返回值说明

返回该数据类型所占的bytes的大小。

## 约束说明

无。

## 调用示例

```python
pypto.bytes_of(pypto.DT_FP32)
```

结果示例如下：

```python
输出: 4
```
