## 功能说明

OpType定义了比较操作的类型，用于指定符号标量之间进行比较运算时的操作类型。主要用于条件判断和逻辑运算中。

## 原型定义

```python
class OpType(enum.Enum):
     EQ = ...  # 等于操作，判断两个值是否相等
     NE = ...  # 不等于操作，判断两个值是否不相等
     LT = ...  # 小于操作，判断左值是否小于右值
     LE = ...  # 小于等于操作，判断左值是否小于或等于右值
     GT = ...  # 大于操作，判断左值是否大于右值
     GE = ...  # 大于等于操作，判断左值是否大于或等于右值
```
